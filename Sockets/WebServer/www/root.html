You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /cat -- returns a picture of one of my cats
    This query has a parameter of Kitty - Range[1-2]
    Example: /cat?kitty=1 will return a picture of my cat Stinky. There are only two cats that can be shown so only numbers 1 or 2 will show an image
  • /ft_to_cm -- returns the height inputted to centimeters
    This query has 2 parameters
    • ft: This parameter takes in an integer value for Feet
    • in: This parameter takes in an integer value for Inches
    Example: /ft_to_height?ft=5&in=10 will return this height converted to centimeters
